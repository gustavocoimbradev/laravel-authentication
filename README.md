# Laravel Authentication

A basic user authentication system built with Laravel. Created for portfolio purposes to practice auth flow and form validation.

## Features

- User registration  
- User login and logout  
- Form validation  
- Tailwind CSS styling  

## Technologies

- Laravel  
- PHP  
- Tailwind CSS  
- SQLite
